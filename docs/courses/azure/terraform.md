# IaC with Terraform

The first link is a short course on Microsoft Learn and will give you an
overview. Afterwards, work with the Fundamentals Lab to have a hands-on
experience with Terraform on Azure. Throughout the lab you will also use Azure
Verified Modules, for which the third link will give you a more in-depth look
into topic and additionally also a follow-up quickstart course.

- [Fundamentals of Terraform on Azure][course-tf-intro]
- [Terraform on Azure Fundamentals Lab][tf-on-azure-lab]
- [Azure Verified Modules][avm]
